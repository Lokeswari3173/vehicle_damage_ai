
# 🚗 Vehicle Damage AI

## AI-Powered Vehicle Damage Detection & Repair Cost Estimation

An intelligent AI-powered web application that analyzes vehicle images to detect damage, estimate severity, and generate repair recommendations using Computer Vision and Vision-Language Models.

---

# 📌 Problem Statement

Vehicle damage assessment is traditionally performed manually by insurance agents and mechanics, making the process:

- Time-consuming
- Subjective
- Costly
- Prone to human error

An automated AI solution can significantly improve inspection speed and consistency.

---

# 🎯 Objective

Develop an intelligent vehicle inspection system that can:

- Detect damaged vehicle parts
- Estimate damage severity
- Generate repair suggestions
- Estimate repair costs
- Provide instant AI-powered reports

---

# 🧠 System Overview

The application combines:

- YOLOv8 Object Detection
- Vision-Language AI Model
- Streamlit Web Interface

Users simply upload an image of a damaged vehicle, and the system generates an AI-based damage assessment report.
